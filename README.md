### What's this?

This is a project repo for DADA of Group2 in 2026 NUS SoC Summer Workshop. It contains our codes & reports when completing the project.

### What's the project about?

The title of the project is "Low-Level Protocol Exploitation and Physical-Layer Defenses for NFC Smart Lock Systems". Our target is to simulate a smart door lock and use high frequency NFC cards to access it. Then, we want to attack it without communicate with the lock, and successfully sniff the right key code when a normal user unlock the door. After attacking, we suppose to defend this, at least, keep our NFC lock away from danger of sniffing.
